# Django Calculator

[View Live Preview](https://django-based-calculator.herokuapp.com/api)

## What it is?

- Basic calculator REST APIs developed in Django
- Routes
  - `/api/sum/<num_1>/<num_2>`
  - `/api/difference/<num_1>/<num_2>`
  - `/api/multiplication/<num_1>/<num_2>`
  - `/api/division/<num_1>/<num_2>`
